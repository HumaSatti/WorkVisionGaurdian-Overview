# WorkVisionGaurdian-Overview
**WorkVisionGuardian** is an AI driven, privacy first workplace productivity and analytics platform.

**What it does**
- Real time activity recognition (e.g., working, idle, on phone, customer service)
- Business KPI correlation and secure dashboards
- Designed for privacy: anonymization, encryption, and explicit consent workflows

**Tech highlights**
Python • PyTorch • OpenCV • YOLO / DeepFace (investigation only) • FastAPI • React • Flutter • Docker • Kubernetes • GitHub Actions

**Security & compliance**
Built with zero trust, least privilege, and data minimization principles. GDPR-aware and consent first for biometric features.
